1. Mobile App Development Fundamentals
   Java / Kotlin (Android) or Swift (iOS) – depending on the platform. Since you’re using Java/Groovy, focus on Android.
   Android Studio – IDE basics, project structure, emulators, building APKs.
   Activities & Fragments – screens and navigation in Android.
   Layouts & Views – LinearLayout, RelativeLayout, ConstraintLayout.
   Event Handling – detecting taps, drags, and gestures.
   Custom Views & Canvas Drawing – for drawing shapes, lines, and connectors (essential for flowcharts).
2. Graphics & Drawing Concepts
   Canvas API – for custom drawing in Android.
   Paint & Path classes – for lines, curves, and shapes.
   Touch Events – drag and drop shapes, connecting nodes.
   Zooming & Panning – for large diagrams on small screens.
3. Flowchart & Diagram Concepts
   Basic Flowchart Symbols – process, decision, start/end, input/output.
   Node & Connector Logic – how shapes link logically.
   Diagram Layouts – hierarchical, sequential, network-style arrangements.
   Editing Features – move, resize, delete nodes, undo/redo.
4. Data Management
   Saving & Loading Diagrams – using JSON or XML for storing node positions and connections.
   Local Storage – SharedPreferences or SQLite/Room database.
   Exporting – save as PNG, JPEG, or PDF.
5. UX/UI Design
   Touch-Friendly UI – buttons, menus, toolbars optimized for mobile.
   Drag & Drop UX – intuitive placement and connection of nodes.
   Responsive Design – works on different screen sizes and orientations.
6. Optional Advanced Concepts
   Undo/Redo Stack – for editing actions.
   Multi-Select & Grouping – manipulate multiple nodes at once.
   Animations – smooth transitions, connection animations.
   Cloud Sync (Optional) – store diagrams online for backup or sharing.
